# 🩺 Disease Prediction from Medical Data

This project is a Machine Learning-based system that predicts whether a disease is **Cancerous or Non-Cancerous** using medical data. It also includes an interactive **Streamlit web application** for real-time predictions.

---

## 🚀 Live Application
🌐 **Try the app here:**  
👉 https://disease-prediction-from-medical-data.streamlit.app/
---

## 📌 Project Overview

The goal of this project is to:
- Analyze medical data
- Perform data preprocessing and cleaning
- Apply Machine Learning algorithms
- Predict disease outcomes accurately
- Provide a user-friendly interface using Streamlit

---

## ⚙️ Features

✔️ Data Cleaning & Preprocessing  
✔️ Exploratory Data Analysis (EDA)  
✔️ Machine Learning Model Training  
✔️ Disease Prediction (Cancerous / Non-Cancerous)  
✔️ Interactive Streamlit Web App  
✔️ Real-time Prediction from User Input  

---

## 🧠 Machine Learning Workflow

1. Data Collection  
2. Data Cleaning  
3. Feature Selection  
4. Model Training  
5. Model Evaluation  
6. Prediction  

---
